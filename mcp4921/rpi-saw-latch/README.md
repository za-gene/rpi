# pi c

Produce a 440Hz sine wave.

## Connection

```
MCP INFO PI
1   3V3   1
2   CS   24
3   SCK  23
4   MOSI 19
5   GND   6
6   3V3   1
7   GND   6
8   VOUT  -

COLS:
* MCP - pin connection to MCP4921
* INFO - description. VOUT goes to amp, not Pi
* PI - physical pin for Pi connection
```
